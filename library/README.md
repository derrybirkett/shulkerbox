# Library

Reference materials, documentation, and resources for building better skills, agents, hooks, and workflows.

## Purpose

This is a curated collection of:
- Official documentation for tools we integrate with
- Best practices and patterns
- Research and articles on productivity systems
- Examples and inspirations
- Technical references

**Curation principle**: Be selective. Only add resources that directly inform how we build or improve this repository.

## Adding Resources

When adding a new resource, consider:

1. **Relevance**: Does this directly help us build better skills/agents/workflows?
2. **Quality**: Is this authoritative, well-maintained, or insightful?
3. **Durability**: Will this be useful in 6 months?
4. **Actionability**: Can we apply learnings from this?

If yes to most of these, add it to the appropriate file with:
- Clear title
- Brief description of what it is
- Why it's valuable (what we learn/apply from it)
- Link

**Remember**: Library bloat is just as bad as code bloat. Be lean, be selective.
